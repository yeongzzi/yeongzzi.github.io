## 문제 설명

학교 학생 3명의 정수 번호를 더했을 때 0이 되면 3명의 학생은 삼총사라고 합니다. 학생들의 번호를 나타내는 정수 배열 number가 매개변수로 주어질 때, 학생들 중 삼총사를 만들 수 있는 방법의 수를 return 하도록 solution 함수를 완성하세요.

**입출력 예 **

| number                   | result |
| ------------------------ | ------ |
| [-2, 3, 0, 2, -5]        | 2      |
| [-3, -2, -1, 0, 1, 2, 3] | 5      |
| [-1, 1, -1, 1]           | 0      |
